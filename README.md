## Установка и запуск

First be sure you have Node.js and npm installed.
Во-вторых запустите mongod.exe.
Для загрузки тестовых данных в БД воспользуйтесь setup.js

    $ git clone https://github.com/igorbronowicki/insider
    $ cd insider
    $ npm install
    $ node app.js
